# kappersproject
Dit project is een website voor een kapperszaak. 

Via de site kan een klant een afpsraak maken of een account aanmaken om afspraken in het verleden in te zien.
Ook kan de klant op de site reviews van andere klanten lezen, behandelingen bekijken en zien wat voor mooie creaties deze kapperszaak eerder al bij klanten heeft weten neer te zetten.

De eigenaar van de kapperszaak en haar medewerkers kunnen via de site zien welke afspraken er op de planning staan of zelf een afspraak toevoegen aan de agenda.

Dit project is gemaakt door Melvin Korving (FE & product owner), Joep van Dam (FE), Milan van der Linden (BA en scrummaster), Dustin van Norden (TI) en Tobias van Dorp (FE)
